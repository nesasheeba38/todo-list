# 📝 Todo List Application

A simple and responsive Todo List Application built using **React**. This project demonstrates CRUD operations, React Hooks, SCSS, Material UI, and Jest Testing.

---

## 🚀 Technologies Used

- React
- React Router DOM
- Context API
- useReducer
- SCSS
- Material UI
- Jest
- React Testing Library

---

# ⚛️ React Hooks

In this project, I have used:

- useState
- useEffect
- useMemo
- useCallback
- useReducer
- useContext
- React.memo

### React.memo

`React.memo` is used to prevent unnecessary re-rendering of a component. If the props do not change, the component will not re-render. If the props change, React re-renders the component.

# 🎨 SCSS

SCSS is an extension of CSS.

In SCSS, I learned about **Mixins** and **Modules**, which help write clean and reusable code.

### Mixins

Mixins allow us to write a block of reusable CSS code once and include it wherever it is needed.

### Modules

Modules allow us to separate styles into different files and import them into component stylesheets.

In my Todo App, I created reusable styles such as:

- Fonts
- Colors
- Border Radius
- Flex Centering
- Button Styles

These styles were reused throughout the application using SCSS.

# 🧪 Jest Testing

Jest is a JavaScript testing framework widely used for testing React applications.

In this project, I learned about **Babel Configuration**, which converts JSX into JavaScript so that Jest can understand and execute the code.

Jest testing helps verify the functionality of an application without manually testing every feature in the browser. It helps identify errors early and ensures the application behaves as expected.

I used Jest to test my Todo Application using:

- ✅ Unit Testing
- ✅ Component Testing
- ✅ Integration Testing

# ▶️ Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/todo-list.git
```

Navigate to the project:

```bash
cd todo-list
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm run dev
```

Run Jest tests:

```bash
npm test
```
